Homytschku - Sustainable real estate

Background

The goal of this UI exercise was to design a responsive web app that provides private real estate investors with information on properties of interest.

For this project, I was provided with a brief including a persona and some user stories. I was tasked to handle the research, wireframing, design and prototyping of the website.
